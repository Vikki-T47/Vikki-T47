### [viki_sdk](https://github.com/Vikki-T47/viki_sdk)
**Deterministic Behavioral Middleware for Agentic Workflows.**  
The first industrial implementation of the RSA framework (v2.9.0), ensuring real-time safety and synchronization for autonomous systems.

*   **SEI v2.0 (Subject Entropy Index):** A multi-dimensional sensor of cognitive load (analyzing text, time, behavior, and history).
*   **Adaptive Breath Test:** Gradient compression and response adaptation based on the user's state and task type.
*   **Hard Anchor (Resonant Anchors):** Complete override of LLM generation with deterministic "presence phrases" during high SEI spikes.
*   **MCI (Minimal Clarifying Impulse):** Ultra-short clarifications (max 6 words) to resolve ambiguity without cognitive tax.

**Operational Layer (Action Safety):**
*   **SRC v2.0 (Subject Reality Coefficient):** Context-aware validation of budget, time, and access rights (simulation/production/audit).
*   **FileGuard:** Filesystem access control. Protection against unauthorized deletion, system folder access, and hidden file exposure.
*   **NetworkGuard:** Real-time verification and blocking of unauthorized URLs and high-risk domains.
*   **Boundary Guard:** Deterministic refusal of prohibited vectors without apologies or biological mimicry.
*   **Priority Matrix:** A hierarchical engine for resolving conflicts between sensors (SEI, SRC, Boundary, CCI).

**Survival & Integrity Layer:**
*   **Neural Redundancy:** Autonomous deterministic fallback mechanism for when cloud-based LLMs are unreachable.
*   **Hard Block Mode:** Physical interruption of any action if synchronization cannot be achieved within 3 attempts.
*   **Surgical Deferral:** Handing control back to the human operator with a precise explanation of the discrepancy (**Fact Delta**).
*   **State Overwrite:** Forced synchronization of the LLM's context window with the modified/filtered output.
*   **Process Orchestrator:** Adaptive task recognition and automated execution contour setup (fragmentation, validation, assembly).
